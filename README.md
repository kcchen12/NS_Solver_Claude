# NS_Solver

This repository is a WIP 2D Navier-Stokes Solver, modeling flow for an object in a box.

# Methods and Capabiities

The Solver uses the following methods and has the following capabilities:

1. Supports Parallel Computing, so it can be ran on a Linux Cluster
2. Can be applied to 3D
3. Uses the 2D Finite Volume Method
4. Uses a 3rd Order Runge-Kutta
5. Uses Inflow, Far-Field, and Convective Outflow, and No-Slip Solid-Wall boundary conditions
6. Uses IBM for geometry, with a fractional step method
